# Gifshoot — 開発ロードマップ

## 概要

Gyazo的なGIF録画ツール。ローカルファースト・OSS。
録画 → HTML管理 → エディター（トリム・モザイク・スタンプ）の3層構成。

---

## Phase 1 — 範囲選択録画（基盤）

**目標:** 範囲をドラッグ選択して最大15秒のGIFを録画する

### タスク
- [ ] 半透明オーバーレイウィンドウを全画面に表示
- [ ] マウスドラッグで録画範囲を選択（ゴムバンド矩形）
- [ ] 選択確定後にカウントダウン表示（3・2・1）
- [ ] 選択範囲のみキャプチャ（mss の region 指定）
- [ ] 最大15秒で自動停止
- [ ] FPS・最大秒数を config.py で設定可能に
- [ ] 録画中はトレイアイコンを赤くする
- [ ] GIFをローカル保存（~/Pictures/Gifshoot/）

### 使用ライブラリ
- `tkinter` — 選択オーバーレイUI
- `mss` — 画面キャプチャ
- `Pillow` — GIF生成

### 完了条件
ホットキーを押したら範囲選択できて、GIFが保存される。

---

## Phase 2 — HTML管理ビューア

**目標:** 撮ったGIFをHTMLファイルで一覧・管理できる

### タスク
- [ ] GIF保存時に `index.html` を自動生成・更新
- [ ] サムネイルグリッド表示（日付降順）
- [ ] 各GIFにファイル名・撮影日時・ファイルサイズを表示
- [ ] GIFホバーでアニメーション再生
- [ ] クリックでエディターを開く（Phase 3 へ）
- [ ] 削除ボタン（確認ダイアログ付き）
- [ ] HTMLは単一ファイル完結（外部CSS・JS不要）

### 完了条件
保存フォルダの `index.html` をブラウザで開くと全GIFが見られる。

---

## Phase 3 — GIFエディター

**目標:** GIFをブラウザ上で編集できる

### 3-1 タイムライン・トリム
- [ ] フレームをタイムライン表示
- [ ] 始点・終点スライダーで範囲指定
- [ ] プレビューをリアルタイム更新
- [ ] トリム結果を新しいGIFとして保存

### 3-2 モザイク
- [ ] 任意の矩形範囲を選択
- [ ] ブロックサイズ選択（S / M / L）
- [ ] 全フレームに一括適用
- [ ] 部分フレームにのみ適用も可能

### 3-3 スタンプ
- [ ] デフォルトスタンプ数枚を同梱
- [ ] ユーザーが画像をアップロードしてスタンプ追加
- [ ] ドラッグで位置調整
- [ ] サイズ変更（角のハンドル）
- [ ] 適用するフレーム範囲を指定可能

### 使用技術
- Canvas API — フレーム描画・編集
- `gif.js` または `gifenc` — ブラウザ上でGIF再生成
- 全処理クライアントサイドのみ（サーバー不要）

### 完了条件
ブラウザ上でトリム・モザイク・スタンプを適用して保存できる。

## 技術メモ

| 項目 | 選択 | 理由 |
|------|------|------|
| 範囲選択UI | tkinter | 標準ライブラリ、依存追加なし |
| キャプチャ | mss | 高速、クロスプラットフォーム |
| GIF生成（Python側） | Pillow | シンプル |
| GIF生成（ブラウザ側） | gifenc | 軽量、wasm不要 |
| エディターUI | Vanilla JS + Canvas | フレームワーク不要 |
