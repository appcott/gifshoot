# Gifshoot 🎯

Free, local-first GIF screen recorder. No account. No cloud. No paywall.

## Quick Start (Windows)

1. Make sure [Python](https://www.python.org/downloads/) is installed
2. Double-click `win_start.bat`
3. Press `Ctrl+Shift+G` to start recording
4. Press `Ctrl+Shift+G` again to stop and save

GIFs are saved to `~/Pictures/Gifshoot/`

## Hotkeys

| Action | Hotkey |
|--------|--------|
| Start / Stop | `Ctrl+Shift+G` |
| Cancel | `Ctrl+Shift+X` |

## Configuration

Edit `config.py` to change:

- Save folder
- FPS (default: 10)
- Max recording length (default: 30s)
- Hotkeys

## Requirements

- Python 3.8+
- Windows (Mac/Linux PRs welcome 🙌)
